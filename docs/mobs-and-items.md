## Usage

- To copy graphics to another app, use the PNGs in `tmp/processed/` for mobs and `summon/summon/MobImages/` for items. These are already resized for the iOS UI.
- If you need the `.imageset` structure for Xcode asset catalogs, see `summon/summon/Assets.xcassets/Mobs/` where imagesets are organized under `Hostile`, `Passive`, and `Items`.

If you want, I can also:
- produce a zip of the `tmp/processed/` folder for easy transfer, or
- generate `.imageset` folders for each processed PNG ready to drop into an Xcode asset catalog.
